# 都道府県別人口データ（2020年）

## 概要

本データは、2020年国勢調査に基づく日本の都道府県別人口データを  
RDF（Turtle形式）としてLOD化したものです。

## データ内容
- 対象：日本の47都道府県
- 人口：2020年時点の総人口
- 形式：Turtle（.ttl）
- 文字コード：UTF-8

## ファイル説明
- pref_population_2020.ttl  
  都道府県ごとの人口データをRDFで記述したファイル

## 語彙・名前空間
- 独自名前空間：`http://example.org/pref/`
- rdf: http://www.w3.org/1999/02/22-rdf-syntax-ns#
- xsd: http://www.w3.org/2001/XMLSchema#
